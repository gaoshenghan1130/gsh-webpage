## Overview

This project is still on **Ongoing**. For updates, please visit the GitHub repository:  
ðŸ‘‰ [Unicycle Project on GitHub](https://github.com/gaoshenghan1130/Unicycle)

This project focuses on developing an autonomous unicycle capable of balancing itself using advanced control algorithms and sensor fusion techniques. To achieve this, this project contains multiple modules for different aspects of the unicycle's functionality, including:

- **Core**: STM32WB55RG, responsible for overall control communication and processing.
- **Sensor**: IMU for orientation and acceleration sensing.
- **Lateral Controller**: Responsible for maintaining balance and stability during lateral movements.
- **Longitudinal Controller**: Responsible for maintaining balance and stability during longitudinal movements.
- **Bluetooth Module**: Handles communication with PC or mobile devices for general direction and control.
- **Communication Module**: Manages data exchange between different components and external devices.

Some of these modules could be found on the child projects on the right bar, including their specific implementations and configurations.
